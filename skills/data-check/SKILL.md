# 목적
정리된 맛집 정보가 믿을 수 있는지 확인한다.

# 확인 항목
- 영업시간 형식이 맞는지
- 지금 영업 중 여부가 맞게 표시되는지
- 가게 이름, 주소, 종류, 영업시간, 링크 중 빠진 것 있는지
- 같은 가게가 두 번 나오는지
- 링크가 열리는지

# 순서
1. 정리된 맛집 데이터 파일을 읽는다
2. 영업시간 형식 점검
3. 링크 열리는지 점검
4. 중복 가게 점검
5. 문제 개수를 센다
6. 심각도별로 정리한다

# 심각도
- 심각: 화면 자체가 안 되는 문제
- 보통: 링크 깨짐, 정보 누락, 영업 여부 틀림
- 경미: 표시가 조금 어긋난 것

# 결과 형식
1. 전체 요약
2. 심각/보통/경미 문제 목록
3. 문제 개수 표
4. 당장 고쳐야 할 것 3개
