# Goal
핵심 기능은 유지하면서 맛집 목록 서비스의 사용성을 점검한다.

# Checklist
- 카드 정보 계층이 명확한가 (가게명 > 카테고리 > 평점 > 영업시간)
- 영업 중 뱃지가 한눈에 들어오는가
- 모바일에서 줄바꿈/overflow 문제가 없는가
- 검색/필터/정렬 UI가 바로 이해되는가
- empty state, loading state, error state가 있는가
- 거리/평점 정렬 전환이 직관적인가

# Procedure
1. 데스크톱 화면 점검
2. 모바일 폭 점검
3. 필터 적용/해제 흐름 점검
4. 검색 결과가 0건일 때 상태 점검
5. 수정 우선순위 제안

# Deliverable
- UI 이슈 목록
- 우선순위 1~3
- 수정 제안
