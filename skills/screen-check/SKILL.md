# 목적
핵심 기능은 유지하면서 화면이 쓰기 편한지 확인한다.

# 확인 항목
- 카드에서 가게명, 종류, 평점, 영업시간 순서가 자연스러운가
- 영업 중 표시가 한눈에 보이는가
- 핸드폰에서 봤을 때 글자나 버튼이 잘리지 않는가
- 검색, 필터, 정렬 버튼이 바로 이해되는가
- 결과 없을 때, 로딩 중일 때, 오류 났을 때 각각 다른 화면이 있는가
- 거리순/평점순 전환이 자연스러운가

# 순서
1. 컴퓨터 화면에서 점검
2. 핸드폰 화면 크기로 점검
3. 필터 켜고 끄는 흐름 점검
4. 검색 결과가 없을 때 점검
5. 고칠 것 우선순위 제안

# 결과물
- 화면 문제 목록
- 우선순위 1~3
- 수정 제안
