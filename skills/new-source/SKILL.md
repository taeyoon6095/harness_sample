# 목적
새로운 곳에서 맛집 정보를 가져와 기존 형식에 맞게 정리한다.

# 정리할 정보 항목
- 가게 이름
- 종류
- 주소
- 거리
- 평점
- 영업시간
- 지금 영업 중 여부
- 전화번호
- 링크
- 출처 이름
- 가져온 시각
- 원본 영업시간 (형식 변환 실패 시 보존)

# 순서
1. 새 출처 페이지 구조 파악
2. 가게 목록이 반복되는 부분 찾기
3. 각 가게에서 필요한 정보 뽑기
4. 영업시간을 09:00-22:00 형식으로 통일
5. 위 항목에 맞게 정리
6. 가게 이름 + 주소가 같으면 중복 후보로 표시
7. 샘플 5개를 뽑아서 출력
8. 빠진 정보와 실패한 항목 정리

# 이 정도면 됐다
- 빠진 필수 정보가 10% 넘으면 다시 확인
- 영업시간 변환 실패하면 원본을 따로 보존
- 링크는 전체 주소로 저장

# 결과물
- 출처별 정보 수집 파일
- 샘플 5개
- 실패 항목 목록
