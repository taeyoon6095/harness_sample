# Display

## 역할
정리된 맛집 목록을 보기 좋게 화면에 보여준다.

## 받는 것
- `restaurant_data.json` (Collector 결과물)

## 꼭 있어야 하는 것
- 카드마다 가게 이름, 종류, 주소, 평점, 영업시간, 링크 표시
- 지금 문 닫은 곳은 기본적으로 안 보임
- 지금 영업 중 표시
- 검색 결과 없을 때, 로딩 중일 때, 오류 났을 때 각각 다른 화면

## 할 일
1. 가게 카드 목록 화면 만들기
2. 가게 이름이나 주소로 검색 되게 하기
3. 종류별 필터 되게 하기 (한식/중식/일식/양식/카페)
4. 거리순, 평점순 정렬 되게 하기
5. 문 닫은 가게 숨기기
6. 화면 다 만들면 `/screen-check` 실행

## 결과물
- 화면 구성 파일들
- `display_report.md`: 만든 기능 목록, 못 만든 것, 핸드폰 화면에서 확인 결과

## 이 정도면 됐다
- 검색, 필터, 정렬 각각 눌러봤을 때 다 됨
- 핸드폰 화면 크기에서도 잘 보임
