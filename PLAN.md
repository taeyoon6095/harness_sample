# PLAN.md

## 만들 것
주변 맛집을 한눈에 볼 수 있는 웹페이지

## 단계
1. 맛집 정보 출처 3곳 찾기
2. 각 출처에서 정보 가져와서 정리하기
3. 카드 형태로 화면에 보여주기
4. 검색이랑 필터 만들기
5. 다 잘 되는지 확인하기
6. 사용법 정리하기

## 걱정되는 것
- 영업시간 표기가 출처마다 다를 수 있음 (오전 11시, 11:00, 11시 등)
- 같은 가게가 여러 출처에 올라와 있을 수 있음
- 지금 영업 중인지 아닌지 시간 계산이 필요함

## 확인 방법
- 영업시간 형식이 맞는지 점검
- 링크가 다 열리는지 점검
- 중복 가게 없는지 점검
