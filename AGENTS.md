# AGENTS.md

## 이 서비스가 하는 일
오늘 뭐 먹을지 고민될 때, 주변 맛집을 한눈에 볼 수 있게 해준다.

## 누가 쓰는가
- 점심 메뉴 고르기 귀찮은 사람
- 특정 음식 종류로 빠르게 찾고 싶은 사람
- 지금 문 연 곳만 보고 싶은 사람

## 담당자
| 담당 | 파일 | 하는 일 |
|---|---|---|
| Finder | agents/finder.md | 맛집 정보 출처 찾기 |
| Collector | agents/collector.md | 정보 모아서 정리하기 |
| Display | agents/display.md | 화면에 보여주기 |
| Checker | agents/checker.md | 잘 됐는지 확인하기 |

## 정보 기준
- 가게 이름, 주소, 종류, 영업시간은 반드시 있어야 한다
- 같은 가게가 두 번 나오면 하나만 남긴다
- 지금 문 닫은 곳은 기본적으로 안 보여준다
- 영업시간은 오전 9시 → 09:00 형식으로 통일한다

## 작업 순서
1. Finder가 먼저 출처를 찾는다
2. Collector와 Display는 동시에 작업 가능
3. Checker는 모두 끝난 후 마지막에 확인한다

## 다 됐다고 할 수 있는 기준
- 맛집 정보 출처 3곳 이상 연결됨
- 검색, 종류 필터, 정렬이 된다
- 문 닫은 곳은 안 보인다
- 같은 가게 중복 없다
- 링크 눌렀을 때 다 열린다
